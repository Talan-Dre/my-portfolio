# My Portfolio Core Repositories

A unified repository containing production-ready microservice layers, mobile application frameworks, and structured database scripts.

## Project Directories

* **kivy_app/**: Standalone cross-platform user interface engine running on Python 3 and Kivy.
* **streamlit_infra/**: Web application cluster wrapped inside a multi-stage execution Docker container.
* **rdbms_layer/**: Normalized relational database schema layout engineered for T-SQL (Microsoft SQL Server).
